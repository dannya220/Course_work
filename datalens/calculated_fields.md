# Расчётные поля DataLens

В дашборде использовались подготовленные дневные данные из папки `data/processed/`. Часть показателей была рассчитана заранее в Python, а часть могла использоваться в DataLens как расчётные поля или поля для группировки.

## Спред доходности к ключевой ставке

Показатель используется для оценки отклонения доходности ОФЗ от ключевой ставки Банка России.

```text
Доходность ОФЗ - Ключевая ставка
```

Экономический смысл: положительное значение означает, что доходность ОФЗ выше ключевой ставки, отрицательное значение — что доходность ниже ключевой ставки.

## Группа срочности

Поле используется для разделения выпусков ОФЗ на группы по сроку до погашения.

```text
до 1 года
1 - 3 года
3 - 7 лет
7+ лет
```

## Тип облигации

Поле используется для сравнения облигаций с постоянным и переменным купоном.

```text
ОФЗ-ПД
ОФЗ-ПК
```
