# Анализ и визуализация данных с использованием Yandex DataLens, на примере данных Московской Биржи

Репозиторий содержит материалы курсовой работы, посвящённой анализу рынка облигаций федерального займа и связи показателей ОФЗ с ключевой ставкой Банка России.

В проекте рассматриваются облигации федерального займа с постоянным купоном (ОФЗ-ПД) и облигации с переменным купоном (ОФЗ-ПК). Основное внимание уделяется динамике доходности, цены, объёма торгов, спреда доходности к ключевой ставке, а также сравнению поведения разных типов облигаций в периоды изменения денежно-кредитной политики.

## Цель проекта

Цель проекта — собрать, обработать и проанализировать дневные данные по рынку ОФЗ за период с 03.05.2016 по 03.05.2026, а также подготовить данные для построения интерактивного дашборда в Yandex DataLens.

В рамках работы решаются следующие задачи:

1. загрузить исторические данные по выбранным выпускам ОФЗ;
2. получить данные по ключевой ставке Банка России;
3. выполнить предварительную обработку данных;
4. подготовить дневные датасеты для анализа и визуализации;
5. рассчитать дополнительные показатели, включая спред доходности к ключевой ставке;
6. построить аналитические графики в Python;
7. подготовить интерактивные страницы дашборда в Yandex DataLens;
8. сравнить поведение ОФЗ-ПД и ОФЗ-ПК.

## Используемые данные

В проекте используются следующие источники данных:

- исторические данные по выпускам ОФЗ, полученные через ISS API Московской биржи;
- данные по ключевой ставке Банка России;
- подготовленные дневные датасеты для построения визуализаций в Yandex DataLens.

Период анализа:

```text
03.05.2016 — 03.05.2026
```

## Описание папок

### `data/raw/`

Папка содержит исходные данные, полученные из внешних источников.

В неё входят:

- исходные CSV-файлы по выбранным выпускам ОФЗ;
- данные по ключевой ставке Банка России.

### `data/processed/`

Папка содержит очищенные и подготовленные дневные датасеты, которые используются для анализа и загрузки в Yandex DataLens.

Основные файлы:

```text

ofz_for_datalens_daily_2016_2026.csv
ofz_fixed_for_datalens_daily_2016_2026.csv
ofz_floating_for_datalens_daily_2016_2026.csv
ofz_fixed_with_key_rate_for_datalens.csv
ofz_floating_with_key_rate_for_datalens.csv
```

### `data/summary/`

Папка содержит итоговые таблицы и промежуточные отчёты, полученные в ходе анализа данных.

Здесь могут храниться:

- сводная таблица по выборке;
- таблица с количеством наблюдений;
- таблица с периодом наблюдений;
- таблицы по отдельным типам ОФЗ;
- отчёты по пропускам и обработке выбросов.

### `notebooks/`

Папка содержит Jupyter Notebook-файлы. Каждый ноутбук отвечает за отдельный этап проекта.

| Ноутбук | Назначение |

## Основные этапы работы

### 1. Загрузка данных по ОФЗ

На первом этапе загружаются исторические данные по выбранным выпускам облигаций федерального займа. Данные сохраняются отдельно по каждому выпуску.

### 2. Подготовка данных по ключевой ставке

Данные по ключевой ставке Банка России загружаются и приводятся к формату, удобному для объединения с дневными данными по ОФЗ.

### 3. Предварительная обработка данных

На этапе предварительной обработки выполняются:

- приведение дат к единому формату;
- объединение данных по отдельным выпускам;
- проверка и удаление дубликатов;
- проверка пропущенных значений;
- обработка аномальных наблюдений;
- разделение данных по типу облигаций;
- подготовка признаков для анализа.

### 4. Подготовка данных для DataLens

После очистки данных формируются дневные CSV-файлы, которые используются для построения интерактивного дашборда.

На этом этапе рассчитываются дополнительные показатели:

- спред доходности к ключевой ставке;
- срок до погашения;
- тип облигации;
- показатели для сравнения ОФЗ-ПД и ОФЗ-ПК.

### 5. Аналитика и визуализация

В аналитических ноутбуках строятся графики, которые позволяют изучить:

- динамику средней доходности ОФЗ;
- динамику ключевой ставки Банка России;
- изменение спреда доходности к ключевой ставке;
- поведение отдельных выпусков ОФЗ;
- зависимость спреда от срока до погашения;
- различия между ОФЗ-ПД и ОФЗ-ПК.

## Как запустить проект

### 1. Клонировать репозиторий

```bash
git clone https://github.com/dannya220/Course_work.git
cd Course_work
```

### 2. Создать виртуальное окружение

Для macOS/Linux:

```bash
python -m venv .venv
source .venv/bin/activate
```

Для Windows:

```bash
python -m venv .venv
.venv\Scripts\activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить ноутбуки

Ноутбуки рекомендуется запускать последовательно:

```text
01_download_ofz_data.ipynb
02_prepare_key_rate_data.ipynb
03_clean_ofz_data.ipynb
04_prepare_datalens_datasets.ipynb
05_market_overview_analysis.ipynb
06_individual_ofz_analysis.ipynb
07_spread_and_maturity_analysis.ipynb
08_fixed_vs_floating_analysis.ipynb
09_summary_tables.ipynb
```

Порядок запуска важен, так как последующие ноутбуки используют файлы, созданные на предыдущих этапах.

## Результаты

В результате работы были получены очищенные дневные датасеты по ОФЗ, рассчитаны дополнительные показатели и построены визуализации для анализа рынка облигаций федерального займа.

Основные содержательные выводы:

- доходность ОФЗ связана с динамикой ключевой ставки Банка России;
- ОФЗ-ПД сильнее реагируют на изменение рыночных ставок через изменение цены и доходности;
- ОФЗ-ПК в большей степени сохраняют цену около номинала, а их реакция на изменение ключевой ставки проявляется через изменение купонных выплат;
- спред доходности к ключевой ставке позволяет оценивать отклонение рыночной доходности ОФЗ от уровня ключевой ставки;
- срок до погашения является важным фактором при анализе чувствительности облигаций к изменению процентных ставок.
