# Порядок запуска ноутбуков

Ноутбуки разбиты по принципу: один ноутбук — одна задача.

Рекомендуемый порядок запуска:

1. `01_download_ofz_data.ipynb` — загрузка дневных исторических данных по выбранным выпускам ОФЗ через ISS API Московской биржи.
2. `02_prepare_key_rate_data.ipynb` — скачивание ключевой ставки Банка России через SOAP-сервис ЦБ РФ и сохранение файла `data/processed/key_rate_2016_2026.csv`.
3. `03_clean_ofz_data.ipynb` — очистка дневных данных ОФЗ, проверка пропусков, дубликатов и выбросов.
4. `04_prepare_datalens_datasets.ipynb` — подготовка дневных CSV-файлов для Yandex DataLens и присоединение ключевой ставки.
5. `05_market_overview_analysis.ipynb` — общий обзор рынка ОФЗ по дневным данным.
6. `06_individual_ofz_analysis.ipynb` — анализ отдельных выпусков ОФЗ.
7. `07_spread_and_maturity_analysis.ipynb` — анализ спреда доходности к ключевой ставке и срока до погашения.
8. `08_fixed_vs_floating_analysis.ipynb` — сравнение ОФЗ-ПД и ОФЗ-ПК.
9. `09_summary_tables.ipynb` — итоговые таблицы для курсовой работы.

В этом варианте проекта месячные датасеты не создаются и не сохраняются. Все выходные таблицы для DataLens строятся на дневных данных.
