# Ссылки на дашборд Yandex DataLens

В рамках курсовой работы был построен интерактивный дашборд в Yandex DataLens. Дашборд состоит из четырёх страниц, каждая из которых отвечает за отдельный аналитический блок.

## Страница 1. Общий обзор рынка ОФЗ

Ссылка:

```text
https://datalens.yandex/p1hngxhqp3ry9
```

Страница используется для анализа общей динамики рынка ОФЗ: средней доходности, ключевой ставки Банка России, спреда доходности к ключевой ставке и объёма торгов.

## Страница 2. Анализ отдельных выпусков ОФЗ

Ссылка:

```text
https://datalens.yandex/zb3crcugwrruj
```

Страница предназначена для анализа отдельных выпусков облигаций. На ней можно рассмотреть динамику цены, доходности и других показателей по выбранному выпуску ОФЗ.

## Страница 3. Анализ спредов и срочности

Ссылка:

```text
https://datalens.yandex/gssqzq6rxghc0
```

Страница используется для анализа спреда доходности к ключевой ставке Банка России и связи этого показателя со сроком до погашения облигаций.

## Страница 4. Сравнение ОФЗ-ПД и ОФЗ-ПК

Ссылка:

```text
https://datalens.yandex/q2wej8hrh0gga
```

Страница предназначена для сравнения облигаций с постоянным купоном и облигаций с переменным купоном. Основной акцент сделан на различии в поведении цены и доходности при изменении ключевой ставки.
