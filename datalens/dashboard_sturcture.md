# Структура дашборда Yandex DataLens

Интерактивный дашборд построен в Yandex DataLens на основе подготовленных дневных CSV-файлов из папки `data/processed/`.

Дашборд состоит из четырёх страниц.

## Страница 1. Общий обзор рынка ОФЗ

Страница предназначена для анализа общей динамики рынка облигаций федерального займа.

Основные элементы страницы:

- динамика средней доходности ОФЗ и ключевой ставки Банка России;
- динамика среднего спреда доходности к ключевой ставке;
- динамика общего объёма торгов.

Используемые данные:

- дневные данные по ОФЗ;
- данные по ключевой ставке Банка России;
- подготовленные файлы из папки `data/processed/`.

## Страница 2. Анализ отдельных выпусков ОФЗ-ПД

Страница предназначена для анализа конкретных выпусков облигаций федерального займа.

Основные элементы страницы:

- динамика цены выбранного выпуска;
- динамика доходности выбранного выпуска;
- показатели по выбранному выпуску;
- сопоставление динамики выпуска с ключевой ставкой.

Используемые данные:

- дневные данные по отдельным выпускам ОФЗ;
- подготовленный датасет для DataLens.

## Страница 3. Анализ спредов и срочности

Страница используется для анализа отклонения доходности ОФЗ от ключевой ставки Банка России.

Основные элементы страницы:

- динамика спреда доходности к ключевой ставке;
- сравнение спреда по группам срочности;
- зависимость спреда от срока до погашения.

Используемые данные:

- дневные данные по ОФЗ;
- рассчитанный спред доходности к ключевой ставке;
- срок до погашения облигации.

## Страница 4. Анализ отдельных выпусков ОФЗ-ПК

Страница предназначена для анализа ОФЗ-ПК.

Основные элементы страницы:

- динамика отклонения от номинала;
- динамика НКД;
- динамика оценочной купонной доходности и ставки ЦБ;
- объем торгов выбранной ОФЗ-ПК
- динамика оценочной купонной ставки;

Используемые данные:

- дневные данные по ОФЗ-ПК;
- данные по ключевой ставке Банка России.
